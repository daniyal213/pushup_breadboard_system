# pushup_breadboard_system
This is the project that helps people to do sports fun and effective.

What is it
It is a pushups breadboard with software that monitors progress. 

It's specific functions
- Count push ups
- Set daily plans(daily quest) and rise it if its done with ease
- Record progress
- Record max push ups for each muscle group

How it works
It consist of two parts hardware(push up breadboard with some electronics) and software(webserver).
